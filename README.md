### 🍇 1 일 1 포도알고리즘 풀기 챌린지 🍇

이트루 알고리즘이랑 친해지기 챌린지의 일환으로 매일 스스로에게 칭찬 스티커라도 부여하며 만족감을 느끼기 위함ㅋㅋㅋ

#### 📌 셀프 포도알 규칙

- ⏰ 30분 룰 : 시간 초과하면 스탑하기
- 📝 하루 1문제 : 2개 이상은 놉 (이었지만 주말엔 2개 이상도 괜찮은걸로~)
- 🚀 프로그래머스 1단계부터 시작 + 가끔 2단계

#### 📌 셀프 포도알 규칙

- 🟣 : 문제 해결 + 풀이
- 🟢 : 문제 해결 못함 + 풀이
- 🟡 : 문제만 푼 날
- ⚫️ : 알고리즘 못한 날
